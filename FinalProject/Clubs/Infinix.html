Infinix Club

VISION STATEMENT

To impress the importance of coding in the digital age; there-by showing how it helps to develop creativity, improve problem-solving, logical reasoning and give a deeper understanding in a plethora of languages like python, C++, JavaScript, Swift, SQL and more.

MISSION STATEMENT

1. Develop an interest in coding by making it more interesting by conducting games, doubt-learning and talk sessions, workshops, hackathons and coding competitions.
2. Organize monthly workshops related to Algorithmic programming, App Development, Web designing, Data structures, Game hacking and many more.

OBJECTIVES

1. Engage students in coding and teach them the basic syntax of various programming languages like C, C++, Java, Python and more.
2. Inculcate and foster the culture of programming on the campus.
